无符号整数1的数量

  1. 朴素法
    与无符号整数的位数有关，与1按位&，然后整数右移一位

   func numberOf1(num uint64) int {
    c := 0
    for num > 0 {
        bn := num & 1
        if bn == 1 {
            c++
        }
        num = num >> 1
    }
    return c
}

  1. 快速法

    只与无符号整数中1的个数有关，num =（num-1)+1，num & (num -1)即可去掉num中最右侧的1

   func numberOf12(num uint64) int {
    c := 0
    for num > 0 {
        num = num & (num - 1)
        c++
    }
    return c
}
