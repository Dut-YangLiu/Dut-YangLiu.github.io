无符号整数1的数量

  1. 朴素法
    与无符号整数的位数有关，与1按位&，然后整数右移一位

   func numberOf1(num uint64) int {
    c := 0
    for num > 0 {
        bn := num & 1
        if bn == 1 {
            c++
        }
        num = num >> 1
    }
    return c
}

  1. 快速法

    只与无符号整数中1的个数有关，num =（num-1)+1，num & (num -1)即可去掉num中最右侧的1

   func numberOf12(num uint64) int {
    c := 0
    for num > 0 {
        num = num & (num - 1)
        c++
    }
    return c
}

golang的单例模式

单例模式

单例模式：只有一个对象实例，全局唯一的公共使用的资源或者配置，节省内存，加快对象访问速度。

go语言单例模式

非线程安全实现

结构体和全局变量都是小写，表示私有变量，只能在该package中使用全局变量
标识是否有初始化，如果没有初始化则通过New方法的调用实现全局初始化，
完成实例的创建，但是上面的方法不是线程安全的。

线程安全实现

在新建实例的时候对全局加锁，加锁之后在创建，在此基础上做优化,
加入原子操作，保证只有在某种情况下才会加锁进行对象实例化，
优化访问效率

golang的Once实现安全的单例模式


package singleton

import "sync"

type single struct{
     O interface{}
}

var sin_instance *single = nil
var once sync.Once

func New()*single{
     once.Do(func(){
          sin_instance = new(single)
     })
     return sin_instance
}

Android 安装 Linux

Posted on 2019-12-16
Symbols count in article: 677 Reading time ≈ 1 mins.

材料准备：

  1. 需要软件：busybox，手机vnc客户端， 终端模拟器
  2. Android设备：获得root权限
  3. linux版本：可以到http://sourceforge.net/projects/linuxonandroid/files/?source=navbar 选择
  4. 需要文件：ubuntu.sh ： ubuntu安装脚本
    bootscript.sh：每次启动ubuntu脚本
  5. 我装的是ubuntu 13.10 small 版，所需文件
    http://pan.baidu.com/s/1mhsofPY

安装步骤：

  1. 安装busybox，手机vnc客户端， 终端模拟器

  2. 将下载的ubuntu.sh bootscript.sh 以及ubuntu.img和ubuntu.md5 放到新建目录ubuntu下面，将ubuntu文件夹存储到android设备下面的内部存储卡(sdcard)或者外部存储卡(extsdcard),我选择了外部存储卡

  3. 打开终端模拟器，su 切换到root模式

  4. 输入命令：cd /mnt/extSdCard/ubuntu，然后 ls -l 查看一下文件是否齐全

  5. 输入命令: sh ubuntu.sh 会提示你输入一些密码，或者一些其他的问题(输入y)

  6. 默认的用户是ubuntu，所以安装完成之后 输入：su ubuntu 切换到ubuntu 用户下，输入 vncpasswd (远程登录的密码)，这样服务端就设置好了。不要关闭会话端口。
