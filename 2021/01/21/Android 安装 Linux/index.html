Android 安装 Linux

Posted on 2021-01-21 Edited on 2021-09-14 Views:
Symbols count in article: 677 Reading time ≈ 1 mins.

材料准备：

  1. 需要软件：busybox，手机vnc客户端， 终端模拟器
  2. Android设备：获得root权限
  3. linux版本：可以到http://sourceforge.net/projects/linuxonandroid/files/?source=navbar 选择
  4. 需要文件：ubuntu.sh ： ubuntu安装脚本
         bootscript.sh：每次启动ubuntu脚本
  5. 我装的是ubuntu 13.10 small 版，所需文件
    http://pan.baidu.com/s/1mhsofPY

安装步骤：

  1. 安装busybox，手机vnc客户端， 终端模拟器
  2. 将下载的ubuntu.sh bootscript.sh 以及ubuntu.img和ubuntu.md5 放到新建目录ubuntu下面，将ubuntu文件夹存储到android设备下面的内部存储卡(sdcard)或者外部存储卡(extsdcard),我选择了外部存储卡
  3. 打开终端模拟器，su 切换到root模式
  4. 输入命令：cd /mnt/extSdCard/ubuntu，然后 ls -l 查看一下文件是否齐全
  5. 输入命令: sh ubuntu.sh 会提示你输入一些密码，或者一些其他的问题(输入y)
  6. 默认的用户是ubuntu，所以安装完成之后 输入：su ubuntu 切换到ubuntu 用户下，输入 vncpasswd (远程登录的密码)，这样服务端就设置好了。不要关闭会话端口。
