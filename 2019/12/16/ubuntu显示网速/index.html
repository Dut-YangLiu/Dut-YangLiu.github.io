  1. 安装依赖项sudo apt-get install python python-psutil python-appindicator
    1. wget下载indicator-sysmonitor：wget -chttps://launchpad.net/indicator-sysmonitor/trunk/4.0/+download/indicator-sysmonitor_0.4.3_all.deb或者使用sudo命令安装
    2. 终端执行：indicator-sysmonitor
    3. 开机自启动，首先创建自启动文件夹mkdir ~/.config/autostart
